Project Iota

Project Iota is an open source community-driven initiative to develop and maintian well documented and standardized industry-grade IoT tools and platform SW and HW components.

All work is published on the project's GitHub site.

Project Iota represents a collection of open-source, patent-free projects licensed under Apache v2.0 License:

  • Electrolink
  • Electrolib
  • Iota HW Platform

Electrolink

Electrolink ia a protocol and SW system based on JSON RPC via MQTT or CBOR RPC via CoAP.

Electrolib

Electrolib is high-level host-side library that gives models and abstractions for various electrical devices, sensors and actuators. It uses Electrolink as the underlaying protocol. Intel MRAA and UPM can be adopted for this purpose

Iota HW Platform

Iota HW Platform are set of BSP (Board Support Packages) for various popular SoCs and boards
