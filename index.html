Project Iota

Project Iota is an open source community-driven initiative to develop and maintian well documented and standardized industry-grade IoT tools and platform SW and HW components.

All work is published on the project's GitHub site.

Project Iota represents a collection of open-source, patent-free projects licensed under Apache v2.0 License:

  • Electrolib
  • Iota Platform
  • uIota Platform

Electrolib

Electrolib is a userspace HAL library used by applications to interract with HW.

It comes in two flavors:

  • Python (electrolib-py)
  • NodeJS (electrolib-js)

Iota Platform

Iota Platform is a SW platform based Linux OS. This is comprised in most cases of U-Boot/Barebox bootloader, Buildroot, Linux Kernel drivers, various configurations, OTA update mechnism and other platform stuff.

Since all Iota Platforms are full-blown Linux computers, this means that it targets are more powerful boards with bigger ammount of RAM and flash - as oposed to uIota Platform.

Iota Platform will support many boards, but we are currently eveluating:

  • Carambola 2
  • Raspbery Pi
  • CHIP
  • WRTNode2

uIota Platform

Iota Platform targets MCUs with very small ammount of RAM and flash memory. The idea is to acheive connectivity, but keep power consumption as low as possible.

uIota Platform should enable programming in:

  • C (via GCC and Makefiles)
  • Micropython
  • Espruino
  • IoT.js
